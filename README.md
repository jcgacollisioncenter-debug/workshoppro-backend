# WorkshopPro Backend - AI Repair Engine

AI-powered automotive repair estimation and management API.

## Features
- AI Damage Analysis (Gemini 2.5 Flash)
- VIN Decoding (NHTSA Integration)
- YieldMax™ Priority Algorithm
- Autonomous AI Learning/Calibration
- Supabase/PostgreSQL Integration
- Secure API Key Management

## Setup
1. `npm install`
2. Configure `.env` with Supabase and Gemini keys.
3. Run `npm run dev`
